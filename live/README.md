# DARKHORSCORE LIVE v4

OBS向け 1920x1080 固定のライブ表示画面です。

## 使い方
1. このフォルダ内の `index.html`, `style.css`, `script.js`, `README.md` を `/live` にアップロード
2. ZIPファイルそのものはアップロードしない
3. GitHub Pagesで `/live/` を開く

## スプレッドシート連携
`script.js` の先頭にある `SHEET_API_URL` にGoogle Apps ScriptのWebアプリURLを入れてください。

対応カラム例：
- 日付
- 競馬場
- レース
- レース名
- 発走
- 締切
- SAFE
- SAFE馬名
- LONGSHOT
- LONGSHOT馬名
- ジヨ
- ジヨ馬名
- メモ
